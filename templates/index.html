PySummary API Documentation

Welcome to the PySummary API! Use the /summarize endpoint to generate text summaries.

Endpoint: /summarize

Submit a POST request with JSON payload containing the text content to be summarized.

Example using cURL:

curl -X POST -H "Content-Type: application/json" -d '{"text": "Your text content goes here."}' http://127.0.0.1:5000/summarize

Optional parameters:

  • length (default: 5): Number of sentences or words in the generated summary
  • focus (default: 'sentences'): Specify 'sentences' or 'words' for summary focus

Response Format

The API responds with a JSON object containing the generated summary:

{"summary": "Generated summary of the text content."}
